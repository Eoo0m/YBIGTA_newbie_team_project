### 🧙 강성우
SICP 라는 책에서는 프로그래머를 마법사로 비유합니다. 컴퓨터의 영혼을 불러내는 존재라는 비유이죠. \
우리는 AI를 다루고 있지만, 비유는 여전히 유효합니다. 완전 멋있지 않나요? 그래서 저는 마법사가 되고 싶습니다. \
마법사가 되고 싶은 강성우라고 합니다.
